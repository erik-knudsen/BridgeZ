Detailed Description

This abstract class implements declaration of generic base strategy for actor classes (local, remote and auto).

Actor strategy classes have a common method interface. The interface is partly defined by means of virtual methods in this class. And partly by means of methods, which are common for all actor managers (Local, Remote and auto). These methods are implemented in this class. Actor strategy classes also share a a number of common signals. These signals are defined here.
