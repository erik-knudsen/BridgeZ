Game Menu Commands

New Session

Use this command to start a new game session.

Shortcuts
Toolbar:   

Deal New Hand

Use this command to deal the next hand in cases where you want to skip the current hand.

Shortcuts
Toolbar:    
Keys:   F1

Activate Deal Profile

Use this command to activate the deal profile. The computer then deals cards according to the given deal profile. For more information about deal profiles see the topic Deal Profile Dialog.

Shortcut
Toolbar:  

Undo

Use this command to undo a bid or play. This is only possible when the computer is standalone or server. One round of bid or play is undone.

shortcut
Toolbar:  
Keys:      F2

Rebid

Use this command to restart the current game with the bidding phase. This is only possible when the computer is standalone or server.

Shortcuts
Toolbar:  
Keys:      F3

Replay

Use this command to to restart the current game with the playing phase. This is only possible when the computer is standalone or server.

Shortcuts
Toolbar:   
Keys:       F4

Claim All

Use this command to claim the remaining tricks.  The claim will not be honored unless it there is strong evidence that your team deserves all of the outstanding tricks.

Shortcuts
Toolbar:  
Keys:     F5

Claim Contract

Use this command to claim just enough tricks to make the contract.  You will not be credited with any overtricks you might otherwise have earned.

Shortcuts
Toolbar:  
Keys:    shift+F5

Concede

Use this command to concede the remaining tricks to your opponents.  Use caution, as the concession cannot be retracted.

Shortcuts
Toolbar:   
Keys:       F6

Hint

Use this command to get a bidding or play hint from the computer.

Shortcut
Toolbar:  
Keys:     space
