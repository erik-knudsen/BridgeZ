Option Menu Commands

Seat Configuration

Use this command to set the configuration of seats etc. Whether players are automatic (played by the computer) or manual. What the names of players are. Whether the computer is standalone, server or client. For more information about seat configuration see the topic Seat Configuration Dialog.

Bidding/Play

Use this command to define the bid and play options. Which conventions are used in the bidding etc.? And what play conventions are used etc. For more information about Bidding/Play see the topic Bidding Dialogs.

Deal

Use this command to define deal options for a deal profile. This profile can be used by the computer to give cards with a given set of characteristica. HCP, number of cards in a given suit etc.. The deal profile can be activated by the "Game" menu command "Activate Deal Profile". For more information about the deal profile see the topic Deal Profile Dialog.

Display Options

Use this command to set certain display options. Card backs, background color etc.. For more information about display options see the topic Display Options Dialog.

Game

Use this command to Set the game options. How the game is played. Is it a game for practice only or do you want to compare your play with how the computer would play the cards. Maybe you want to play a rubber etc.. For more information about game options see the topic Game Options Dialog.
